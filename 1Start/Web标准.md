# Web标准

### 1.概念

用来建立Web网站的技术的规范文档。

> 对于我们学习如何使用这些技术**帮助甚微**
> 主要用来供软件工程师实现这些技术

* [HTML标准](https://html.spec.whatwg.org/multipage/)

### 2.Web标准组织

* HTML语言现代化组织：[WHATWG](https://whatwg.org/)

* 基于JavaScript的ES标准组织：[ECMA](https://www.ecma-international.org/)

* 3D图形技术组织(webGL)：[Khronos](https://www.khronos.org/)
